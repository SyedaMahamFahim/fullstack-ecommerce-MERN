# Run Locally

Clone the project

```bash
  git clone https://github.com/SyedaMahamFahim/fullstack-ecommerce-MERN.git
```

Go to the project directory 

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```


Start the server

```bash
  npm start
```
